# Global eLibrary

A modern online book platform for browsing, purchasing, and reading books digitally.

## Features

- 📚 **Browse Books** - Explore a wide collection of books across different categories
- 🛒 **Shopping Cart** - Add books to cart and manage purchases
- 👤 **User Accounts** - Create an account and manage your purchased books
- 💳 **Secure Checkout** - Process payments securely
- 📖 **Digital Reading** - Access your purchased books anytime
- ⭐ **Ratings & Reviews** - Share your thoughts on books
- 🌙 **Dark Mode** - Toggle between light and dark themes
- 📱 **Responsive Design** - Works seamlessly on desktop, tablet, and mobile

## Pages

- **index.html** - Home page with featured books and promotions
- **browse.html** - Browse and search all available books
- **book.html** - Individual book details page
- **cart.html** - Shopping cart management
- **checkout.html** - Purchase checkout process
- **success.html** - Order confirmation page
- **dashboard.html** - User profile and order history
- **login.html** - User login page
- **register.html** - New user registration

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Framework**: Bootstrap 5
- **Icons**: Font Awesome 6.4
- **Fonts**: Poppins, Playfair Display (Google Fonts)
- **Storage**: LocalStorage for cart and user data

## Installation

1. Clone or download this project
2. Open `index.html` in your web browser
3. No installation or build process required!
